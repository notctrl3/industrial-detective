# Industrial Detective — AI That Solves Root Causes

## Project Overview

Industrial Detective is an AI-powered manufacturing failure root cause analysis system that can:
- Discover hidden correlations and patterns
- Identify root causes to avoid recalls or downtime
- Provide actionable insights and corrective action recommendations
- Provide an intuitive UI for factory operators and analysts

## Tech Stack

### Frontend
- Next.js 14 (React)
- TypeScript
- Tailwind CSS
- Recharts / Plotly (Data Visualization)
- Shadcn/ui (UI Components)

### Backend
- Python Flask
- Pandas (Data Processing)
- Scikit-learn (Machine Learning)
- NumPy (Numerical Computing)

### Data Analysis
- Time series analysis
- Correlation analysis
- Anomaly detection
- Root cause analysis algorithms

## Project Structure

```
industrial-detective/
├── frontend/          # Next.js frontend application
├── backend/           # Python Flask backend
├── data/              # Data files
├── models/            # Machine learning models
└── docs/              # Documentation
```

## Quick Start

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
